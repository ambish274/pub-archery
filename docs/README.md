Banner System
The application features a dynamic banner system that randomly selects and displays an image at the bottom of the screen.

File Requirements: The system is configured for 10 unique banners. Files must be named sequentially from banner1.jpg through banner10.jpg.

Dimensions: For optimal display, banners should be 800px × 133px.

Error Handling: If the application cannot locate the randomly selected file, it will attempt to load banner1.jpg as a default. If banner1.jpg is also missing, the banner display will be bypassed entirely to ensure a smooth user experience.
